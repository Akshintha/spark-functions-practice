# Spark Functions Practice

This project covers basic PySpark DataFrame functions:

- select()
- withColumn()
- filter()
- common Spark functions like col(), lit(), upper(), lower(), when()

## Topics Covered

1. Creating a Spark DataFrame
2. Selecting specific columns
3. Adding new columns using withColumn()
4. Filtering rows using filter()
5. Using Spark built-in functions
